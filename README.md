# Digital Twin of Andrew Ng

An AI-powered Digital Twin of Andrew Ng built using

- Gemini 2.5 Flash
- FastAPI
- React
- ChromaDB
- SQLite
- RAG
- Persistent Memory

## Features

- Retrieval-Augmented Generation
- Long-term Memory
- Timeline Awareness
- Persona Consistency
- Multi-turn Conversations
- Voice Support
- Memory Dashboard
